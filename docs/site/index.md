NetScript is a Deno-native framework for building durable, plugin-driven backends.
This documentation is organised on the [Diátaxis](https://diataxis.fr/) model, which
separates four distinct kinds of documentation so you can find what you need by intent.

- **[Tutorials](/tutorials/)** — learning-oriented lessons that take you from zero to a
  working result, step by step.
- **[How-to guides](/how-to/)** — task-oriented recipes for solving a specific problem.
- **[Reference](/reference/)** — information-oriented API documentation generated from the
  source via `deno doc`.
- **[Explanation](/explanation/)** — understanding-oriented discussion of how and why
  NetScript works the way it does.

New to NetScript? Start with the [tutorials](/tutorials/). Looking up an API? Go straight
to the [reference](/reference/).
