How-to guides are for getting a specific job done. They assume you already know
the basics and want the shortest reliable path from intent to a working change.

If NetScript is new to you, start with the [tutorials](/tutorials/). For exact
API signatures, use the [reference](/reference/). For the concepts behind a
task, read the [explanation](/explanation/) pages.

## Guides

- [Add a first-party plugin](/how-to/add-a-plugin/) — add workers, sagas,
  triggers, or streams to an existing workspace and verify the wiring.
