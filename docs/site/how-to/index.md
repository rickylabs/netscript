How-to guides are **task-oriented**: focused recipes that walk you through solving a
specific, real-world problem with NetScript. They assume you already know the basics — if
you are brand new, start with the [tutorials](/tutorials/). For exact API signatures see the
[reference](/reference/), and for the concepts behind a task read the
[explanation](/explanation/).

## Guides

- [Add a first-party plugin](/how-to/add-a-plugin/) — add workers, sagas, triggers, or
  streams to an existing workspace and verify the wiring.
