## Lessons for Promotion

| Lesson                                                    | Pattern                                                                                     | Applies to                      | Confidence |
| --------------------------------------------------------- | ------------------------------------------------------------------------------------------- | ------------------------------- | ---------- |
| Scratch-app hosting as minimal Fresh harness              | When the real-app harness lives outside the worktree, a run-local scratch app under `.llm/tmp/` is the narrowest reachable Fresh harness. | All frontend-archetype runs    | high       |
| Windows MAX_PATH can mask genuine framework incompatibilities | `ChildProcess.unref`-style errors on deep paths often indicate path length, not library compatibility.                  | Any Deno/Vite/esbuild spike on Windows | high  |
| Root `exclude` can silently block JSR publish graph       | Removing only the package-local folder from root exclude (keeping task-level excludes intact) unblocks `deno publish --dry-run` without broad-root churn. | Any package publish run        | high       |
| Per-item CSS travel with registry items                   | Extraction is move-only; renames, if any, belong in a later run.                              | Registry/archetype-4 runs      | medium     |
| OKLCH re-derivation preserves consumer contract           | `--ns-*` invariants survive CSS fallback + OKLCH declaration coexistence.                     | Design-system runs             | high       |
