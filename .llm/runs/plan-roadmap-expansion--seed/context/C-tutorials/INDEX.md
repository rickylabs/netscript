# context/C-tutorials/ — B4 output

Notes, sketches, and open questions for Topic C, useful to Fable but not fitting B1–B3. Evidence
for every claim below lives in `analysis/C-tutorials/` and `research/C-tutorials/`; this folder does
not re-derive it.

## Files in this folder

- `candidate-tutorial-mappings.md` — evidence-based options for which real project backs each of the
  4 rewrites and the shape of the minimal eis-chat tutorial. Lays out Options A/B/C and 2 shape
  candidates; does **not** choose (per topic-C §4, that decision is delegated to Fable).
- `open-questions.md` — 7 open questions for Fable/Stage C, including the highest-confidence one
  (Q1: whether `chat` counts as one of "the 4" given it shipped after the historical 4-track
  decision).
- `drift-candidates.md` — 5 spec-vs-repo-reality contradictions (D1–D5), each with an evidence chain:
  the "4 vs 5 tracks" drift, issue #232's actual disjoint scope, tutorial-chapter nav blast-radius
  into 8 capability hubs, the missing beta.6/beta.7 milestones, and the domain-literal-vs-discipline-
  literal ambiguity in "follow the eis-chat approach."

## How to read this folder

Start with `drift-candidates.md` (what's actually contradictory) → `open-questions.md` (what
therefore needs a decision) → `candidate-tutorial-mappings.md` (the options once those decisions are
made). None of the three should be read as a recommendation to act on without Fable's ratification.
