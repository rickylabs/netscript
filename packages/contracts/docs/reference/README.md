# Reference

## Root

`@netscript/contracts` exports the base contract, common error schemas, pagination schemas, schema
helper factories, result types, and diagnostics.

## CRUD

`@netscript/contracts/crud` exports CRUD contract generators.

## Query

`@netscript/contracts/query` exports pagination query helpers, filter schemas, and filter builders.

## Transform

`@netscript/contracts/transform` exports record transformation helpers.
