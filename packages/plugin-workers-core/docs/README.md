# Workers Core Documentation

This directory documents `@netscript/plugin-workers-core`.

## Pages

- `architecture.md` explains the Tier 1 package boundary.
- `concepts.md` defines jobs, tasks, workflows, ports, and presets.
- `getting-started.md` gives a short root-import path.
- `recipes/` contains task-oriented examples.

## Status

The docs describe the core package as implemented through Phase 6a slice D16. Later slices migrate
the Tier 2 plugin and downstream consumers.
