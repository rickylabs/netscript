# Config Concepts

## NetScriptConfig

The validated project configuration object consumed by framework packages.

## PartialConfig

A plugin contribution fragment that can add services, apps, databases, deploy settings, and runtime
config paths without overriding project identity.

## Scaffold Constants

Shared directory, file, and permission names used by generators and plugin authors.

## Plugin Schema Fragment

The appsettings entry schemas for plugin services, background processors, and installed version
metadata.
