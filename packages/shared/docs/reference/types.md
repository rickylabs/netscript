# Types

The shared package exports explicit data types for pagination, success envelopes, errors, schema
contracts, contract builders, and diagnostics.

Key groups:

- `OffsetPaginationInput`, `OffsetPaginationQuery`, `OffsetPaginationMeta`
- `CursorPaginationInput`, `CursorPaginationQuery`, `CursorPaginationMeta`
- `SuccessResponse`
- `SharedSchema`, `SharedObjectSchema`, `SharedParseResult`
- `BaseContract`, `BaseContractRouteOptions`, `BaseContractProcedure`
- `InspectionReport`, `InspectionStatus`, `SharedInspectionTarget`
- `Result`, `OkResult`, `ErrorResult`
