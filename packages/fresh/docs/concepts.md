# Concepts

## Route Contracts

Route contracts describe path params, search params, and typed link helpers used by page builders.

## Deferred Rendering

Deferred rendering lets a page render stable shell content while partial data refreshes in the
background or through Fresh partial requests.

## Forms

Form helpers normalize submission intent, validation errors, idempotency, and response state.

## Streams

Stream helpers bridge Fresh rendering with NetScript stream state and telemetry.

## Testing

The `./testing` subpath provides small fixtures for route contexts and defer policies.
